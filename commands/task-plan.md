ユーザーと対話して子エージェントに渡す仕様を決める。

以下のテンプレートに沿ってユーザーに質問し、仕様をまとめる:

- **目的**: 何を実現したいか
- **対象ファイル/ディレクトリ**: 変更が必要な場所
- **受け入れ条件**: 何ができたら完了か
- **制約**: 使ってはいけないもの、守るべきルール
- **補足**: その他の情報

全項目が埋まったら、まとめた仕様をユーザーに提示して確認を求める。

ユーザーが承認したら仕様を保存:

```bash
node ~/.cursor-power/scripts/save-plan.mjs --content "<まとめた仕様のmarkdown>"
```

保存後、ユーザーに「プラン <ID> を保存しました。/task-add で実行できます。」と伝え、すぐにタスク登録するか聞く。

ユーザーが実行を希望したら:

```bash
node ~/.cursor-power/scripts/add-task.mjs --plan <プランID> --repo "<ワークスペースパス>" --base "<ベースブランチ>"
```

続いて:

```bash
node ~/.cursor-power/scripts/start-worker.mjs --task-id <タスクID>
```

「タスク <ID> を開始しました」と報告する。
